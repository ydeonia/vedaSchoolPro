## ID CARD — WHAT'S NEEDED

### Student ID Card Fields
- School name, logo, motto/tagline
- Student photo (placeholder if none)
- Full name, Class & Section, Roll Number
- Admission Number, DOB
- Blood Group
- Father's name, Emergency contact number
- Address
- School phone/landline
- Barcode/QR with student ID
- Valid from / Valid to (academic year)
- Accreditation badges (CBSE/ICSE affiliation number)
- Principal signature (from uploaded signature)

### Employee ID Card Fields
- School name, logo, motto
- Employee photo
- Full name, Employee ID
- Designation, Department
- Blood Group
- Emergency contact number
- Address
- School phone/landline
- Valid from / Valid to
- Barcode/QR with employee ID
